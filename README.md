# CommerciumContracts - A Fully On-Chain Aggregator

## Installation

Install the requirements (LINUX):

```bash
#Install Protostar
#curl -L https://raw.githubusercontent.com/software-mansion/protostar/master/install.sh | bash -s -- -v $PROTOSTAR_VERSION
make setup
```

## Compile Contracts

```bash
make build
```

## Generate Interfaces

```bash
make gen-interfaces
```

## Test

```bash
# Run all tests
make test
